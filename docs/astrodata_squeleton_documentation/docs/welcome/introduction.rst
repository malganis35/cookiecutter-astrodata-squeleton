Introduction
============

Welcome to the **AstroData Squeleton**!

Philosophy
----------

The AstroData Squeleton is more than just a set of folders. It is a **standardized environment** designed to help Data Scientists and Machine Learning Engineers move from experimentation to production with confidence.

Key Benefits:
-------------

*   **Reproducibility**: Powered by `uv` for lightning-fast, deterministic dependency management.
*   **Production-Ready**: Built-in FastAPI for serving models and Streamlit for data exploration.
*   **Quality Gates**: Automated linting and type checking (Ruff, Mypy) pre-configured.
*   **Observability**: Centralized logging and standardized path management.
*   **Traceability**: Docker images with embedded OCI labels for full metadata tracking.

Why use this squeleton?
------------------------

In high-stakes data environments, the biggest bottleneck isn't the model—it's the **infrastructure and engineering** around it. This project solves the "it works on my machine" problem by providing a common language and structure for the whole team.

Whether you are building a simple analysis or a complex real-time API, AstroData Squeleton provides the right foundations.
